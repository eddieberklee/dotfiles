dotfiles
========

Repository set up to store all my settings files such as .rc's, linux config files, plain text files describing changes I always make to new computers, etc. Not only limited to dotfiles.

## Terminal

- [iTerm2 tmux window movement](iterm/README.md): installable `Ctrl` + `Shift` + `Command` + `H`/`L` bindings that move a tmux window while keeping it selected.
- [Codex hooks](ai/codex/hooks.json) and [shared tmux agent state](rc/tmux-agent-state): give Codex the same 🔔 attention and ✅ completion styling as Claude Code, rolled up from panes to the tmux window tab. Run [link-codex.sh](ai/codex/link-codex.sh) to install the global Codex files.
